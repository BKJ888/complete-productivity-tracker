# 🎯 生產力追蹤系統

一個結合 **番茄工作法、習慣追蹤、週間反思** 的 Web App，支援本地儲存 (LocalStorage)。

## 功能
- 🍅 番茄鐘：任務管理與時間追蹤
- 📅 每日追蹤：建立習慣並記錄
- 📊 週間反思：完成率、挑戰、改善計劃
- 📈 月度報告：數據分析與視覺化 (Chart.js)
- 🎯 長期目標管理
- 🏅 成就系統（徽章解鎖）
- 🌙 深色模式切換

## 使用方法
1. 打開 `index.html`  
2. GitHub Pages 部署後，直接透過瀏覽器即可使用  
3. 所有數據存於本地瀏覽器 (LocalStorage)
